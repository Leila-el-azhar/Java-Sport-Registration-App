# Sports Registration Manager (Java)

**Description:**  
A Java application designed to manage registrations for sports activities, track payments, and send notifications to users.

**Features:**  
- User registration and sport selection  
- Payment tracking (paid / unpaid)  
- Notification of upcoming events based on the selected sport  
- Automated payment reminders for unpaid users  
- Implemented using Object-Oriented Programming (OOP), conditional logic, and event management  

**Technologies:**  
- Java  
- OOP principles (classes, inheritance, encapsulation)  
- Event management and conditional logic
